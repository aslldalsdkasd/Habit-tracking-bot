# VK-BOT трекинга привычек

## О БОТЕ

### Бот трекинга привычек помогает бросить вредные привычки. <br>Привычка считается брошенной если 21 день вы воздержались от привычки, <br> в случае не выполнение привычки все дни этой привычки скидываются. <br> Бот оповестит что вы набрали 21 день, <br> также присылает оповещения что нужно отметить выполненные и невыполненные привычки в 20.00. <br> Бот сохраняет все данные в таблице при выключении, при запуске данные и логи не теряются

## НАСТРОЙКА ПЕРЕД ЗАПУСКОМ

##### В ENV.TEMPLATE написаны основные компоненты:<br> BOT_TOKEN - ключ бота вк, <br>GROUP_ID- номер группы в VK,<br> DATABASE_URL_ALEMBIC- URL для подключения алембика(синхронный), <br> DATABASE_URL- основной URL для подключения к бд, <br> DATABASE_URL_CELERY- URL для подключения CELERY к бд.<br> Перед началом нужно создать DATABASE таблицы код создает сам при запуске контейнера. ############

# ЗАПУСК VK-BOT

#### VK-BOT запускается через docker-compose up --build. <br> БД закрывает все подключения перед выключением
